## Деплой
1. Клонуй репо
2. Встанови `python -m venv .venv && source .venv/bin/activate`
3. Встанови залежності `pip install -r requirements.txt`
4. Створи `.env` з `BOT_TOKEN`
5. Запусти `python bot.py`
